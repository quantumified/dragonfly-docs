How to update sector data

This section provides instructions on how to update sector data in the application.

Updating Data

To update the sector data, use the `/updatedata` command. You need to provide the following options:

  • data: The times and sectors data. This should be a space-separated list of times and sector information.
  • timezone: The timezone of the provided data. Select from the available timezones.

Example command:

/updatedata data="12:00 SectorA 14:00 SectorB" timezone="America/New_York"

Viewing Upcoming Sectors

Use the `/nextsector` command to view the next 7 upcoming sectors. This command does not require any parameters.

Example command:

/nextsector

Getting Help

If you need more information or assistance, use the `/help` command to get instructions on how to use the bot commands.

Example command:

/help
